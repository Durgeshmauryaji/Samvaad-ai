# ⚡ Samvaad AI

Samvaad AI is a ChatGPT-like conversational AI assistant built using Groq LLM, LangChain Agents, Firebase Authentication, and Streamlit.

It provides real-time AI responses with Google Search integration, streaming output, persistent multi-chat history, and a clean interactive UI.

---

## 🚀 Live Demo

🔗https://samvaad-ai.streamlit.app

---

## ✨ Features

- 🔐 User Authentication (Signup/Login)
- 💬 ChatGPT-style Multiple Chat Sidebar
- 🗂 Persistent Chat History using Firebase
- ⚡ Real-time Streaming Responses
- 🌐 Google Search Tool Integration
- 🤖 Groq LLM Integration
- 🧠 AI Agent using LangChain
- 🗑 Delete Chats
- 🏷 Auto Chat Titles
- 🎨 Interactive Streamlit UI
